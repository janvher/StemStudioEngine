# GameManager Usage Examples

Practical examples demonstrating GameManager APIs in Studio 3D behaviors.

## Files

- **input_example.js** - Direct keyboard and mouse event handling
- **physics_example.js** - Apply forces, impulses, and manage physics bodies
- **animation_example.js** - Play, blend, and control animations
- **audio_example.js** - Load and play 2D/3D audio clips
- **collision_example.js** - Distance and physics-based collision detection
- **object_picker_example.js** - Click and hover detection
- **flight_controller_example.js** - Complete flight physics controller using direct input events
- **fps_controller_example.js** - First-person shooter controller using direct input events
- **vehicle_controller_example.js** - Car/vehicle physics controller using direct input events
- **point_click_controller_example.js** - Top-down point-and-click movement controller using PointerEventManager
- **weapon_system_example.js** - Weapon behavior with fire rate, raycast, and damage system using direct input events
- **inventory_system_example.js** - Inventory system with pickup and item management
- **parallax_scroller.js** - Parallax enviroment sroller example

⚠️ **These examples use direct browser events (keyboard, mouse, pointer) for clear and reliable input handling.**
